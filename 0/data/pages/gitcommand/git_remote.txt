-----
==== Git remote ==== 
 

 
  * 如何修改远端git仓库 remote 的url地址？

<code bash>
$ git remote set-url remote URL
</code>

----
 ==== 添加一个远端仓库到一个已经存在的git仓库上？====
<code bash>
$ git remote add --track master origin  https://github.com/user/repo.git
</code>
----
==== 设置跟踪远端分支和取消跟踪远端分支====
<code>
$ git remote set-url remote URL
#设置跟踪远端分支
$ git remote set-url remote URL
#取消跟踪远端分支
</code>
